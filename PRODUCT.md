# Product

## Register

product

## Users

Russian-speaking C++ learners who use the site as a local course reader. They move between lessons, compare Russian explanations with original English wording, and need the interface to stay readable while they study dense technical material.

## Product Purpose

Provide a local Russian learning shell for LearnCpp-style material: course navigation, lesson pages, original-text hints, code examples, practice blocks, and resilient offline-friendly access while the content is gradually localized.

## Brand Personality

Calm, technical, careful. The product should feel like a dependable study tool, not a marketing page.

## Anti-references

Avoid decorative dashboards, loud landing-page composition, fragile layouts that require horizontal scrolling, and hidden failure states when local data cannot load.

## Design Principles

- Reading comes first: prose, code, and navigation must not fight for attention.
- C++ deserves precision: technical terms, code fragments, and annotations should be visibly distinct.
- Local work should be forgiving: server failure should not leave the user staring at an empty page.
- Navigation must stay close: current lesson, surrounding lessons, and next steps should be easy to reach.

## Accessibility & Inclusion

Target readable contrast, keyboard-friendly controls, visible focus states, reduced-motion-safe transitions, and layouts that remain usable on mobile screens and local file openings.
