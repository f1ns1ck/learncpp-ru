# Learn C++ RU

Локальная русская оболочка для изучения C++ с оглавлением, поиском, страницами уроков и подсказками оригинальных английских терминов.

## Что внутри

- `src/` — Vue-приложение: компоненты страниц, загрузка данных и небольшие утилиты рендера текста.
- `styles.css` — визуальная система, типографика, адаптивность и состояния интерфейса.
- `scripts/content.js` — встроенные данные интерфейса и частично локализованный учебный контент.
- `learncpp-toc.ru.json` — русифицированное оглавление курса.
- `public/learncpp-toc.ru.json` — копия оглавления, доступная Vite dev/build как статический файл.
- `index.html` и `lesson.html` — точки входа приложения.

## Запуск

```powershell
.\start-local.ps1
```

Скрипт установит npm-зависимости при первом запуске и откроет Vite dev-сервер на свободном порту из диапазона `8123-8125`.

Ручной вариант:

```powershell
npm install
npm run dev -- --port 8123
```

После запуска:

- Главная: `http://127.0.0.1:8123/index.html`
- Урок: `http://127.0.0.1:8123/lesson.html?id=cpp-basics`

## Сборка

```powershell
npm run build
npm run preview
```

## Как добавлять локализацию

1. Обновите источник оглавления в `learncpp-toc.raw.json`.
2. Обновите русские метаданные уроков в `learncpp-toc.ru.json` и синхронизируйте копию в `public/learncpp-toc.ru.json`.
3. Для связи перевода с оригиналом сохраняйте поля `ruTitle`, `title`, `originalTitle`.
4. Полные тексты уроков добавляйте в `scripts/content.js`, если у вас есть право на локальное хранение и перевод материалов.

Полные оригинальные материалы LearnCpp.com добавляйте только при наличии прав на их перевод и локальное хранение.
